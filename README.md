# Library Management System

## 📌 About the Project
This project is a **Library Management System** developed using **PHP and MySQL**.  
It was created to practice **CRUD operations**, database integration and basic back-end development.

The system allows users to manage books, including adding, viewing, updating and deleting records from the database.

---

## 🚀 Features
- Register new books
- View the list of books
- Update book information
- Delete books from the system
- Connection with MySQL database
- Basic CRUD operations

---

## 🛠️ Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP (Local server)

---

## 🎯 Purpose of the Project
The goal of this project was to:

- Practice **back-end development with PHP**
- Learn how to connect **PHP with MySQL**
- Understand **CRUD operations**
- Improve basic **database management skills**
